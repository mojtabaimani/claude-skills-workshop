# Claude Skills Workshop

A hands-on workshop for learning how to build custom **Skills** (slash commands) for [Claude Code](https://docs.anthropic.com/en/docs/claude-code). Includes presentation slides, a live demo app, and a set of ready-to-use example skills.

---

## What Is This Workshop About?

Claude Code Skills let you define reusable, on-demand workflows as markdown files (`SKILL.md`). Instead of re-explaining a complex task every session, you write it once and invoke it with a slash command like `/standup` or `/invoice`.

This workshop covers:

- What Skills are and how they differ from `CLAUDE.md`, hooks, and permissions
- The anatomy of a `SKILL.md` file (frontmatter + body)
- Skill scopes and discovery
- Dynamic context injection and supporting files
- Advanced patterns and best practices
- Hands-on exercises using a real calculator app

## Workshop Materials

### Slides (`slides.md` / `slides.html`)

A full workshop deck built with [Marp](https://marp.app/). Topics covered:

1. What are Skills?
2. Skills vs other Claude Code features (`CLAUDE.md`, auto memory, hooks, permissions)
3. Creating your first skill — file structure and minimal example
4. `SKILL.md` anatomy — frontmatter and body
5. Skill scopes and discovery
6. Dynamic context injection
7. Supporting files and composition
8. Advanced patterns
9. Best practices
10. Hands-on exercises

Open `slides.html` in any browser to view the rendered deck, or use `npx @marp-team/marp-cli slides.md` to re-render.

### Calculator App (`calculator.html`)

A single-page calculator built with vanilla HTML, CSS, and JavaScript — no build tools or dependencies. It serves as the live demo app for the workshop's hands-on exercises.

**Features:**
- Basic arithmetic: `+`, `-`, `×`, `÷`, `%`
- AC (clear all) and DEL (backspace)
- Decimal support, chained operations
- Negate (`±`) and square (`x²`) buttons
- Full keyboard support

Open `calculator.html` directly in any modern browser to use it. See `calculator.md` for the full feature reference and keyboard shortcuts.

## Skills

All skills live under `.claude/skills/`. Each is a directory containing a `SKILL.md` file with a YAML frontmatter header (`name`, `description`) and step-by-step instructions for Claude to follow.

### `hello`
**Greet the user and summarize the current project.**
Reads the project's config file (e.g., `package.json`, `pyproject.toml`) and outputs a brief summary of what the project does, available commands, and the primary language/framework.

### `standup`
**Summarize git commits from the last 24 hours as a standup update.**
Runs `git log --since="24 hours ago"`, groups related commits, and formats the output as a clean standup report with a task count summary at the end.

### `add-button`
**Add a new button to the calculator app with custom function and styling.**
Interactively asks for a button label, description, logic, and color. Then edits `calculator.html` to add the button, updates `calculator.md`, generates a LinkedIn post and an X/Twitter post, and commits everything to git.

### `invoice`
**Generate a professional invoice for a client.**
Asks for client name, hours worked, and hourly rate. Calculates the total and writes a formatted markdown invoice (with invoice number, due date, line items, and payment terms) to a dated file in the current directory.

### `news`
**Get the latest news and updates on any topic.**
Asks for a topic (Technology, Programming, World News, Science, or custom), a preferred source, and how many updates to fetch. Uses web search to retrieve and summarize results as a numbered list with headlines, sources, and one-sentence summaries.

### `tutorial`
**Interactive tutorial that teaches any concept, adjusted to your available time.**
Asks what topic to learn and how much time is available (2, 5, 15, or 30 minutes). Delivers a tailored tutorial — from a quick bullet-point overview to a full deep-dive with exercises — adapted to the current project's language and context.

## Getting Started

1. Clone this repository.
2. Open `slides.html` in your browser to follow along with the workshop.
3. Open `calculator.html` to use the demo app.
4. In Claude Code, invoke any skill with its slash command, e.g.:

```
/standup
/add-button
/tutorial
```

Skills are discovered automatically from the `.claude/skills/` directory.

## Creating Your Own Skill

A minimal skill is just a directory with a `SKILL.md` file:

```
.claude/skills/my-skill/
└── SKILL.md
```

```markdown
---
name: my-skill
description: What this skill does in one sentence
---

# My Skill

Step-by-step instructions for Claude...
```

Invoke it with `/my-skill` in Claude Code.
